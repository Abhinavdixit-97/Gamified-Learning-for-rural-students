## 🆘 Troubleshooting

### Git Not Installed?
Download: https://git-scm.com/download/win

### Script Not Working?
Use VS Code method instead (easier)

### Permission Denied?
Configure git:
```bash
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
```

### Large Files Error?
Check .gitignore includes:
- node_modules/
- dist/
- *.log
